# nodemcu-01

## "Blinkprogram" på NodeMCU
Instruktion för att utföra blink med Arduino IDE och Pulsivo
* Installera Arduino IDE på din dator
* Installera ESP8266-stöd i Arduino IDE
* Anslut NodeMCU till datorn med USB-kabel.
